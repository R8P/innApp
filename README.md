
# innApp

reqres.in api'sini kullanarak kullanıcıları listeleyen, anlık kullanıcı eklenebilen ve kullanıcı detay görüntülenebilen bir uygulamadır. Ayrıca bir adet fake login sayfası bulunmaktadır.


## Yükleme 

```bash 
  yarn install
```
##  ios
```bash
  yarn pod
  yarn run ios
```
## Android
```bash
  android
  yarn run android
```
